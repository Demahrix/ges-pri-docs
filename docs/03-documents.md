Cette page permet de partager des documents avec les autres utilisateurs de la plateforme.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les documents** (`READ_DOCUMENT`).
:::

![Page Gérer (hors ligne)](/img/7.png)
![Page Gérer (hors ligne)](/img/8.png)

## En-tête de la page

- **Champ de recherche** : recherche un document par son titre.
- **Bouton « Filtrer »** : ouvre une fenêtre modale avec les critères suivants :
  - Type de document
  - Domaine thématique
  - Organisation responsable
  - Année
  - Type de fichier
  - Taille (Min)
  - Taille (Max)
  - Enregistré par
  - Date de création (Min)
  - Date de création (Max)

  Trois boutons permettent de valider l'action : **Effacer** (réinitialise les critères), **Fermer** (ferme la fenêtre sans appliquer) et **Rechercher** (applique les filtres).
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des documents

| Colonne | Description |
|---|---|
| Titre | Titre du document |
| Type de document | Type du document |
| Domaine thématique | Domaine thématique associé au document |
| Organisation responsable | Organisation à l'origine du document |
| Année | Année du document |
| Nb. téléchargement | Nombre de fois où le document a été téléchargé |
| Crée le | Date de création (dépôt) du document |
| Crée par | Utilisateur ayant déposé le document |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Télécharger** : télécharge le fichier.
- **Voir** : affiche les informations complémentaires du document — Type de fichier, Taille, Date de la dernière modification — en plus des colonnes déjà visibles dans le tableau.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour les informations du document.
- **Supprimer** : supprime le document.
