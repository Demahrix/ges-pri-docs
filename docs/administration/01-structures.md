# Structures (`/structures`)

Cette page affiche la liste des structures utilisant la plateforme (centres, antennes, associations partenaires, etc.). C'est à l'une de ces structures que sont ensuite associées les fiches d'identification, les PRI et les évaluations.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

![Page Structures](/img/20.png)

## En-tête de la page

- **Champ de recherche** : recherche une structure par son nom.
- **Bouton « Nouveau »** : ouvre le formulaire de création d'une nouvelle structure.
- **Bouton « Filtrer »** : permet de filtrer la liste selon le statut.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des structures

| Colonne | Description |
|---|---|
| # | Numéro de ligne |
| Nom | Nom de la structure |
| Communes | Commune(s) de rattachement de la structure |
| Statut | Statut de la structure |
| Crée le | Date de création |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : affiche le détail de la structure.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour la structure.
- **Supprimer** : supprime la structure.
