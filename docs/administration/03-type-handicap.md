# Types d'handicap (`/type-handicap`)

Cette page affiche la liste des types d'handicap proposés dans les fiches d'identification.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

![Page Type d'handicap](/img/23.png)

## En-tête de la page

- **Champ de recherche** : recherche un type d'handicap par son nom.
- **Bouton « Nouveau »** : ouvre le formulaire de création d'un nouveau type d'handicap.
- **Bouton « Filtrer »** : permet de filtrer la liste selon le statut.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des types d'handicap

| Colonne | Description |
|---|---|
| # | Numéro de ligne |
| Libellé | Nom du type d'handicap |
| Statut | Statut du type d'handicap |
| Crée le | Date de création |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : affiche le détail du type d'handicap.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour le libellé ou le statut.
- **Supprimer** : supprime le type d'handicap.

:::tip
Modifier cette liste change directement les options proposées dans la partie « Handicap et suivi médical » du formulaire de [fiche d'identification](../enfants-et-pri/02-fiche-identification.md).
:::
