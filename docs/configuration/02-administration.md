# Administration

:::info[Permission requise]
L'accès à l'ensemble des pages ci-dessous nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

## Structures (`/structures`)

Liste des structures utilisant la plateforme (centres, antennes, associations partenaires, etc.), chacune rattachée à une Commune. C'est à cette structure que sont ensuite associées les fiches d'identification, les PRI et les évaluations. Permet de consulter, ajouter, modifier ou supprimer une structure.

## Utilisateurs (`/utilisateurs`)

Gestion des comptes utilisateurs de la plateforme : création de comptes, attribution d'un rôle (Facilitateur, Superviseur, Administrateur, etc.) et d'une structure de rattachement, désactivation ou suppression d'un compte, réinitialisation de mot de passe selon les droits disponibles.

## Types d'handicap (`/type-handicap`)

Référentiel des types de handicap proposés dans les fiches d'identification (Handicap moteur, Handicap visuel, Handicap auditif, Handicap intellectuel/déficience mentale, Polyhandicap, Albinisme, Trisomie 21, Infirmité motrice cérébrale, etc.). Permet d'ajouter, modifier ou supprimer un type afin d'adapter la liste proposée dans le formulaire d'identification.
