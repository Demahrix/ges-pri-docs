# Rapports (`/rapports`)

Cette page présente, sous forme détaillée, l'ensemble des données synthétisées de manière résumée sur le [tableau de bord](./dashboard.md) : identifications, PRI et évaluations, avec des possibilités d'analyse par structure, Région, Province, Commune, période, type de handicap, etc.

Elle est accessible à tous les utilisateurs connectés (aucune permission spécifique n'est requise pour son affichage dans le menu), mais les données visibles restent limitées à celles auxquelles votre compte a accès.
