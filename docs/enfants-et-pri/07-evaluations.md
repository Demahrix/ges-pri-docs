# Évaluations (`/evaluations`)

Cette page liste l'ensemble des évaluations de PRI réalisées sur la plateforme.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les évaluations** (`READ_EVALUATION`).
:::

## En-tête de la page

- **Champ de recherche** : recherche un bénéficiaire par son nom.
- **Bouton « Filtrer »** : ouvre une fenêtre modale avec les critères suivants :
  - Structure
  - Commune
  - Village / Secteur
  - Mode de saisine
  - Surnom
  - Sexe
  - Occupation
  - Lieu d'habitation
  - Statut d'orphelin
  - Date de création (Min)
  - Date de création (Max)
  - Enregistré par

  Trois boutons permettent de valider l'action : **Effacer**, **Fermer** et **Rechercher**.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**.

:::note
Comme pour les PRI, une évaluation se crée toujours **à partir d'un PRI existant** (bouton « Faire une évaluation » sur celui-ci).
:::

## Tableau des évaluations

| Colonne | Description |
|---|---|
| Enfant | Nom du bénéficiaire concerné |
| Structure | Structure ayant réalisé l'évaluation |
| Objectifs | Aperçu des objectifs évalués |
| Etat | État de l'évaluation |
| Crée le | Date de création de l'évaluation |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : ouvre la [page de détail](./details-evaluation.md) de l'évaluation.
- **Modifier** : ouvre le formulaire pré-rempli.
- **Supprimer** : supprime l'évaluation.
- **Voir la fiche d'identification** : ouvre la fiche d'identification du bénéficiaire concerné.
- **Voir le PRI** : ouvre le Plan PRI évalué.
