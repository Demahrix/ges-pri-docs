# Détail d'une évaluation (`/evaluations/details/:id`)

Cette page affiche l'ensemble des informations de l'évaluation : en-tête et tableau des objectifs / activités prévues / activités réalisées / résultats atteints / observations.

![Vue de la page details d'evaluation](/img/36.png)

## Bouton « Actions »

- **Voir** *(déjà sur cette page)*
- **Modifier** — ouvre le formulaire pré-rempli.
- **Supprimer** — supprime l'évaluation.
- **Voir la fiche d'identification** — ouvre la fiche d'identification du bénéficiaire.
- **Voir le PRI** — ouvre le PRI évalué.
