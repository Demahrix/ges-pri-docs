# Provinces (`/provinces`)

Cette page affiche la liste des Provinces référencées sur la plateforme.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

![Page Provinces](/img/17.png)

## En-tête de la page

- **Champ de recherche** : recherche une Province par son nom.
- **Bouton « Nouveau »** : ouvre le formulaire de création d'une nouvelle Province.
- **Bouton « Filtrer »** : permet de filtrer la liste selon le statut.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des Provinces

| Colonne | Description |
|---|---|
| # | Numéro de ligne |
| Code | Code de la Province |
| Nom | Nom de la Province |
| Région | Région de rattachement |
| Statut | Statut de la Province |
| Crée le | Date de création |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : affiche le détail de la Province.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour la Province.
- **Supprimer** : supprime la Province.
