# Régions (`/regions`)

Cette page affiche la liste des Régions du Burkina Faso référencées sur la plateforme.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

![Page Régions](/img/16.png)

## En-tête de la page

- **Champ de recherche** : recherche une Région par son nom.
- **Bouton « Nouveau »** : ouvre le formulaire de création d'une nouvelle Région.
- **Bouton « Filtrer »** : permet de filtrer la liste selon le statut.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des Régions

| Colonne | Description |
|---|---|
| # | Numéro de ligne |
| Code | Code de la Région |
| Nom | Nom de la Région |
| Statut | Statut de la Région |
| Crée le | Date de création |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : affiche le détail de la Région.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour la Région.
- **Supprimer** : supprime la Région.
