# Communes (`/communes`)

Cette page affiche la liste des Communes référencées sur la plateforme.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les paramètres d'administration** (`VIEW_ADMINISTRATION_SETTINGS`).
:::

![Page Provinces](/img/18.png)

## En-tête de la page

- **Champ de recherche** : recherche une Commune par son nom.
- **Bouton « Nouveau »** : ouvre le formulaire de création d'une nouvelle Commune.
- **Bouton « Filtrer »** : permet de filtrer la liste selon le statut.
- **Boutons d'export** : **Copier**, **CSV**, **Excel**, **PDF**, pour exporter la liste affichée (en tenant compte des filtres et de la recherche en cours).

## Tableau des Communes

| Colonne | Description |
|---|---|
| # | Numéro de ligne |
| Nom | Nom de la Commune |
| Province | Province de rattachement |
| Région | Région de rattachement |
| Longitude | Longitude géographique de la Commune |
| Latitude | Latitude géographique de la Commune |
| Statut | Statut de la Commune |
| Crée le | Date de création |
| Action | Actions disponibles sur la ligne |

### Actions disponibles

- **Voir** : affiche le détail de la Commune.
- **Modifier** : ouvre le formulaire pré-rempli pour mettre à jour la Commune.
- **Supprimer** : supprime la Commune.
