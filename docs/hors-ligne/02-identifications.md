# Identifications (`/hors-ligne/identifications`)

Cette page affiche les fiches d'identification collectées hors ligne et **pas encore envoyées au serveur**.

:::info[Permission requise]
L'accès à cette page nécessite la permission **Voir les données hors ligne** (`READ_OFFLINE_DATA`).
:::

![Page Identifications (hors ligne)](/img/6.png)

Elle reprend la même structure que la page [Identifications](../enfants-et-pri/identifications.md) : champ de recherche, filtres, tableau des fiches avec les colonnes Nom et prénom(s), Sexe, Age, Handicap, Niv. scolar., Structure, Crée le, et les actions associées (Voir, Modifier, Supprimer).

La différence est que les fiches listées ici sont uniquement celles **en attente d'envoi**, stockées localement sur l'appareil. Une fois envoyées via le bouton **Envoyer en ligne** de la page [Gérer](./gerer.md), elles disparaissent de cette liste et deviennent visibles dans la liste habituelle des [Identifications](../enfants-et-pri/identifications.md).
